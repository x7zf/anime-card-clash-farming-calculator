# Anime Card Clash Hub

Includes:
- Farming Calculator
- Teams Database
- Hidden Admin Panel
- Preset dropdowns with Other/custom support
- Custom enemies/categories become presets
- Image upload, drag/drop, and paste support
- Favicon
- Cloudflare Worker for safely updating `teams.json`

## Admin

Secret command: `accadmin`

Default password in this build is `REPLACE_THIS_PASSWORD`.
Change it before publishing if you keep the frontend password.
Remember: frontend passwords are not real security in a public repo.
